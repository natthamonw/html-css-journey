The World Wide Web is a collection of HTML documents. The internet is the world's largest computer network. When the user connects to the internet, she uses a browser that runs on a computer to display files found on the web. The browser (e.g. Chrome, Safari, Internet Explorer) does this via the HTTP protocol which allows it to communicate with the servers or computers that host the files that make up the web.
