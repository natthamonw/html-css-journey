CSS

Cascading Style Sheets

What is CSS?

CSS is the code writtent to give style to HTML elements. It is possible to have several Cascading Style Sheets referenced by one HTML document. Along with HTML and JavaScript, CSS is used by most websites to create visually engaging webpages, user interfaces for web applications, and user interfaces for many mobile applications.

CSS Format

In the example below, the CSS declaration consists of a selector (h1), one or more properties (background-color) with their value (yellow), followed by a semicolon (;) and flanked by a pair of curly brackets ({}).

h1{background-color: yellow;}

CSS Selectors

It is important to note that in CSS, the most specific rules apply all elements related to those rules.

  • Tag selectors
    • p, em, h1, h2, h3
    • Example: p{color: green;}
    • Tag selectors are most useful when changing properties that are unique to that HTML element.
  • Class selectors
    • .class-name
    • Example: .description{color: yellow;}
    • Class selectors are most useful and versatile
  • ID selectors
    • #id-name
    • Example: #navigation{color: red;}
    • ID selectors are the most specific selectors and more difficult to override

The Box

Margin, Border, and Padding

Possible values for the margin, border and padding include:

  • Pixels (px)
  • Percentage (%) for better user experience

selector{
margin: 30px 30px;
border: red solid 2px;
padding: 10% 10%;
}

Simplifying the Box

Use selector{box-sizing: border-box;} to make the calculation of the element size include both the border and the padding. Note that margin is not included in the size.

Also use browser-specific prefixes to make sure it displays properly across browsers.

-webkit-box-sizing: border-box;
-moz-box-sizing: border-box;
-ms-box-sizing: border-box;

Flex Box

Use selector{display: flex;} to allow <div>s to be positioned right next to each other as opposed to on top of each other.

For safari to display this properly, add the following declaration:

display: -webkit-flex;

Visit the Complete Guide to Flexbox for more information on the topic.

Useful CSS Tips

CSS References

There is no need to memorize the endless number of CSS properties and values. If you forget them or simply want to learn more about CSS, just visit this Mozilla Firefox CSS Reference or simply use our old friend Google!

Normalize

Default style rules sometimes differ between browsers so one can use normalize.css to make all browsers render all elements more consistently.

Verifying CSS

There are multiple CSS validation services out there to help you keep errors at bay. You can use this W3C CSS Validation Service or easily search for a CSS validator on Google.
