Hypertext Markup Language

What is HTML?

HTML or Hypertext Markup Language is a standardized system for tagging text files to achieve font, color, graphic, and hyperlink effects on World Wide Web pages. HTML documents make up the heart of the Web and can do a number of things.

HTML Markups

Markup element includes an opening tag, content, and a closing tag. In the example below, <p> is the opening tag. "HTML is awesome!" is the content. </p> is the closing tag.

<p>HTML is awesome!</p>

In the opening tag of an element, there may be one or more attributes with their values assigned. In the example below, the img element has two attributes: src and alt.

<img src="images/cute-puppies.jpg" alt="Adorable puppies">

HTML Tags

HTML tags can be divided into two categories: inline elements and block elements. Inline elements only take up as much width as they need and does not force new lines while Block elements take up the full width available, with a new line before and after the element.

  • Inline
    • Bold <b></b>
    • Italics <em></em>
    • Anchor <a href="..."></a>
    • *Image <img>
    • *Line break <br>
    • Span <span></span>
  • Block
    • Paragraph <p></p>
    • Div <div></div>

*These tags only have an opening tag and do not require a closing tag. They are called void tags.

Structure of HTML Documents

<!DOCTYPE html>
<html>
<head>
</head>
<body>
</body>
</html>

  • <!DOCTYPE html> is used to tell the browser that this is an HTML document
  • <html> is the opening HTML tag
  • <head> is the opening head tag holding the site title, links to CSS and Javascript, and meta data
  • </head> is the closing head tag
  • <body> is the opening body tag where most of the visible content of the document is
  • </body> is the closing body tag
  • </html> is the closing HTML tag
